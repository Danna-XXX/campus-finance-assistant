# 🐱 小花猫 · 大学生理财搭子


**一句话定位：** 面向大学生的 AI 理财搭子，基于心理账户理论，覆盖消费管控 → 储蓄积累 → 入门理财的全流程闭环。

---

## 产品背景

大学生理财的核心痛点不是"不会理财"，而是三个具体问题：

1. **钱的来源不同，心理意义不同** — 生活费、兼职收入、奖学金不该混在一起管
2. **不想手动记账** — 传统记账 App 要求逐笔输入，难以坚持
3. **需要在消费当下干预** — 月底复盘没用，买之前的那一刻才是关键

小花猫的核心创新：**心理账户体系** — 按钱的来源而非用途来分类管理，AI 不只判断「够不够买」，还判断「该不该从这类钱里买」。

---

## 核心功能

| 模块 | 功能 |
|------|------|
| 💰 收入区 | 三账户余额管理，按日期/类型记录新收入，走势图（基石稳定/自给3色/荣誉稀疏）+ AI 鼓励 |
| 📊 消费记录 | 今日 / 月度 / 季度 / 年度四维度，Treemap 可视化，点选类别查看明细 |
| 🎯 储蓄目标 | 设置短/长期目标，可选资金来源，优先级排序，已完成目标折叠展示，顺序/同时两种储蓄方案 |
| ⏳ 购物冷静区 | 大额冲动消费加入 24 小时冷静清单，AI 分析值不值得买 |
| 💡 理财入门 | 保底资金达标后触发，三阶梯方案（货币基金/定期/基金定投）+ 产品对比 + 平台直达链接 |
| 🔔 资金预警 | 月度结余概览（正负柱状图）、今日预算设置、最低资金阈值进度条 |
| 📊 账单深度分析 | 上传微信支付账单 Excel → 总览 / 月度趋势 / 时段对比 / 消费人格四维报告 |
| 💬 全局 AI 对话 | 自然语言记账/决策/理财咨询，支持三种搭子人设切换 |

## 心理账户体系

| 账户 | Emoji | 来源 | 说明 |
|------|-------|------|------|
| 基石账户 | 🏠 | 父母生活费 | 保障日常生存，固定月收入 |
| 自给账户 | 💪 | 兼职·实习·接单 | 自己挣的钱，代表独立感，支持按类型分色走势图 |
| 荣誉账户 | 🎓 | 奖学金·竞赛奖金 | 成就感来源，支持记录奖学金类型（国家/一等/竞赛等） |

**保底资金：** 不是独立账户，而是一个安全线目标（默认 ¥500）。总余额达标后即可开始考虑理财，比「攒够3倍月收入」更适合大学生。

## 技术栈

- **前端**：[Streamlit](https://streamlit.io)（单文件多路由 + 6个功能区独立对话）
- **LLM**：DeepSeek-V3（通过[硅基流动](https://cloud.siliconflow.cn)调用）
- **账单解析**：openpyxl（微信支付 Excel 格式）+ 两阶段分类（关键词匹配 + LLM 批量补分类）
- **数据可视化**：Plotly（Treemap / 饼图 / 折线 / 堆叠柱状 / 分组柱状）
- **语言**：Python 3.10+

## 本地运行

### 1. 克隆项目

```bash
git clone https://github.com/你的用户名/xiaohuamao.git
cd xiaohuamao
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置 API Key

复制 `.env.example` 为 `.env`，填入你的硅基流动 API Key：

```bash
cp .env.example .env
```

编辑 `.env`：

```
SILICONFLOW_API_KEY=sk-你的真实Key
```

> 或者在 `.streamlit/secrets.toml` 中填入（Streamlit Cloud 部署推荐）：
> ```toml
> SILICONFLOW_API_KEY = "sk-你的真实Key"
> ```

### 4. 启动应用

```bash
streamlit run app.py
```

浏览器打开 `http://localhost:8508`（端口由 `.streamlit/config.toml` 配置，默认 8508）

## 部署到 Streamlit Cloud（生成公网二维码）

1. 将代码推送到 GitHub（`.env` 和 `secrets.toml` 已被 `.gitignore` 排除，不会泄露）
2. 访问 [share.streamlit.io](https://share.streamlit.io)，用 GitHub 账号登录
3. 点 "New app" → 选择仓库 → 主文件填 `app.py`
4. 展开 "Advanced settings" → Secrets 中填入：
   ```toml
   SILICONFLOW_API_KEY = "sk-你的真实Key"
   ```
5. 点击部署，约 1-2 分钟后得到公网地址
6. 用公网地址生成二维码，放入比赛报告

## 文件结构

```
小花猫/
├── app.py              # Streamlit 主应用（路由 + 8 个页面 + 所有 UI 逻辑）
├── agent.py            # LLM 调用 + 意图解析 + 状态更新（apply_action）
├── models.py           # 数据模型（3账户 / 目标 / 消费记录 / 冷静清单 / 挑战）
├── bill_parser.py      # 微信账单解析 + 关键词分类 + LLM 批量补分类
├── prompts.py          # LLM system prompt（人设 / 区域 / 账单分析）
├── config.py           # API 配置（支持 .env 和 Streamlit secrets）
├── demo_data.py        # 预置 Demo 数据 + 收入记录 + 预计算账单分析结果
├── requirements.txt    # 项目依赖
├── sample_bill.xlsx    # 内置示例账单（跨度一年，2025-05 至 2026-04）
├── .env.example        # 环境变量模板（复制为 .env 后填入真实 Key）
├── .gitignore          # 排除 .env 和 secrets.toml
└── .streamlit/
    ├── config.toml     # 本地服务配置（端口 8508）
    └── secrets.toml    # 本地 Streamlit secrets（不提交到 Git）
```

## Demo 体验路径

**落地页两种入口：**

**路径一（快速体验，推荐）：**
1. 打开应用 → 落地页选择搭子人设 → 点击「开始体验」
2. 直接进入主页，所有模块已预置真实感数据（含收入/消费/目标/冷静清单）
3. 左侧 6 个导航按钮：收入区 / 消费记录 / 储蓄目标 / 购物冷静区 / 理财区 / 资金预警
4. 每个功能区均有独立的小花猫对话框（右侧栏）

**路径二（账单导入）：**
1. 打开应用 → 落地页点击「上传账单」
2. 选择内置示例账单（瞬间出结果，无需 LLM 调用）或上传自己的微信账单 `.xlsx`
3. 查看多维报告（总览 / 月度趋势 / 时段对比 / 消费人格）
4. 点击「进入核心功能」→ 账户已根据账单自动配置
